Loadshift Chrome Extension Demo

A mock interface for the Loadshift Chrome Extension designed to help truck drivers find and analyze transport jobs.

1. Main Popup Interface

This is the main popup interface that appears when users click on the Chrome extension icon. It allows truck drivers to:

  • Select from their saved trucks or add a new one
  • Enter transport preferences including origin/destination, date range, and cargo details
  • Select additional preferences like route options
  • Click "Find Perfect Match" to search for suitable transport jobs
Loadshift Assistant

2. Analysis & Transport Plans Page

After clicking "Find Perfect Match", the system analyzes the driver's requirements and displays matching transport plans:

  • Shows a step-by-step progress bar of the AI analysis process
  • Displays a list of recommended transport plans with key details
  • Highlights matching score, estimated profit, and route information
  • Allows users to click "View Details" to see comprehensive analysis of each plan
Loadshift Assistant - Analysis

3. Transport Plan Details Page

The details page showcases the AI-powered analysis for a specific transport plan:

  • Provides basic transport information including shipper, receiver, and cargo details
  • Shows comprehensive cost analysis with breakdown of expenses
  • Displays weather analysis along the route with safety recommendations
  • Provides traffic and duration analysis with optimal route suggestions
  • Offers a summary with AI-driven recommendations and insights
  • Includes a "Place Bid" button that links to the Loadshift website
